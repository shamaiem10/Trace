Trace is a research workspace for examining the data behind a women’s-health model, comparing raw and population-reweighted outputs, and making model limits visible.

✦Who uses Trace? Women’s-health research teams, data scientists, and study leads. It is not for patient assessment, clinical decisions, or diagnosis.
Research model auditEvidence before confidence
How Trace is used

A workspace for checking the model—not judging a person.

The shipped run uses a labelled research source and an NHANES reference extract. Future teams can reproduce this workflow or adapt the pipeline to approved, compatible datasets.

01

Audit the data

Compare the research source with a broader reference population before assuming it is representative.

02

Train two models

Train the original model and a population-reweighted counterpart using the same held-out test set.

03

Measure the shift

See whether population reweighting materially changes model outputs or performance.

04

Publish limits

Share reproducible artifacts, uncertainty, SHAP sensitivity, and validation boundaries.
